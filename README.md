# gods

The more disciples a god has, the more powerful they are.  As a minor deity, your goal is to create a religion that subsumes all others, either by killing the nonbelievers or converting them to yours, the true religion.

## Elements

* **Peaceful**
  * **Philosphy** - Make and convert discliples.  Kill indirectly.
  * **Power** - Defense
* **Violent**
  * **Philosphy** - Kill disciples directly
  * **Power** - Offense, destroy
* **Mystic**
  * **Philosphy** - Opponents disciples continue to exist but are no longer disclipes.  Anti fundamentalist.
  * **Power** - Draw

## Card Types

* Object
  * Belief
  * Creature
    * Disciple
  * Relic
* Event
  * Miracle

## Special Rules

Each player starts with 5 discliples in play.  Those are 0 cost, chosen from their deck.

If damage is unblocked, opponent determines where damage is allocated.  If at end of turn, player controls no disciples they lose.
